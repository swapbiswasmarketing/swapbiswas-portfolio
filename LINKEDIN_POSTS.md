# LinkedIn Post Playbook - swapbiswas.com

## Instructions

### Posting Rules
- **Frequency:** 3-4 posts/week max for best reach. 5/week is possible but each post gets ~20-30% fewer impressions.
- **Best days:** Tuesday, Wednesday, Thursday (highest engagement). Monday is okay. Avoid weekends.
- **Best time:** 8-10 AM IST (your audience is India + global B2B).
- **Format:** Copy everything between the `---` dividers. Paste directly into LinkedIn. Don't change the line breaks - they're intentional for readability.

### Before Posting
- Verify the UTM link works (click it yourself)
- Check that @ tags match real LinkedIn company/person pages. If a tag doesn't resolve, remove the @ and leave the name plain.
- Generate the image using the Nano Banana prompt provided with each post. Download at 1200x628 (LinkedIn optimal). If no custom image, use the blog's OG image as fallback.
- Posts with custom images get 2x engagement vs text-only.

### After Posting
- **First 60 minutes are critical.** Reply to every comment. LinkedIn's algorithm decides reach in the first hour.
- Ask a question back to commenters to keep the thread going.
- Don't edit the post after publishing (kills reach).
- If someone shares it, comment on their share thanking them.

### Tracking
- Mark status as `[POSTED] YYYY-MM-DD` after publishing
- After posting, add the entry to the **Posted Log** table below so you never post the same blog twice
- Check GA4 > Acquisition > Traffic acquisition > filter by `linkedin` after 24-48 hours to see UTM traffic
- Track which posts get the most engagement - double down on those topics
